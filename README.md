# Copilot Workshop
This repository contains step-by-step documentation for integrating Copilot Studio with Databricks Genie.

## Contents

- [docs/agent-steps/Copilot_Studio_Agent_Flow_Setup.md](docs/agent-steps/Copilot_Studio_Agent_Flow_Setup.md): Instructions for setting up agent flows in Copilot Studio.
- [docs/agent-steps/Copilot_Studio_Agent_Setup.md](docs/agent-steps/Copilot_Studio_Agent_Setup.md): Guide for initial agent setup in Copilot Studio.
- [docs/agent-steps/Databricks_Genie.md](docs/agent-steps/Databricks_Genie.md): Steps for integrating Databricks Genie.
- [docs/agent-steps/Integration_and_Publish_to_Teams.md](docs/agent-steps/Integration_and_Publish_to_Teams.md): How to integrate and publish agents to Microsoft Teams.

## Getting Started

1. Clone this repository to your local machine.
2. Navigate to the `docs/agent-steps` directory for detailed setup instructions.
3. Follow each guide in order for a complete agent integration workflow.

## Support

For questions or support, please refer to the documentation in the `docs/agent-steps` folder.
